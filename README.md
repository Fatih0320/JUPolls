# JUPolls

Polls Component for Joomla! 3.x. Fork from free version MijoPoll and AcePoll.

**Structure:**
* Component
* Module
* Search plugin

**Template support:**
* Bootstrap 2.x
* Bootstrap 3.x

**Demo:**
* [Lviv City Council](http://city-adm.lviv.ua/polls)
* [WZ - Newspaper](http://wz.lviv.ua/polls)
